<b>Mart PH0MH:</b> behuizing, draaibankwerkzaamheden voor oscillator behuizing en afscherming van de verzwakker.

<b>Arie PA3A:</b> software ontwerp.

<b>Costo:</b> software, co-ontwerp besturingsprint, documentatie.
